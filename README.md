# Event Reservation System

A web application for managing event reservations built with Symfony 7.

## Features

- Users: Browse events and make reservations
- Admins: Full CRUD on events, view all reservations
- Modern authentication with JWT and Passkeys (WebAuthn)

## Technologies

- Symfony 7.4
- PostgreSQL 15
- JWT Authentication (LexikJWT)
- WebAuthn/Passkeys
- Docker
